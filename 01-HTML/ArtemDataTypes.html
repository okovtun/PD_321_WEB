DataTypes

С++ supports the following data types:

1. Primary data types
Built-in or predefined data types that can be used directly by the user to declare variables.
•integer
•character
•boolean
•floating point
•double floating point
•void
•wide char

2. Derived data types
Derived data types that are derived from the primitive or built-in datatypes.
•function
•array
•pointer
•reference

3. Abstract or User-Defined data types
Are defined by the user itself.
•class
•structure
•union
•enumeration
•typedef

Primitive Data Types

Integer. Storing integer data.
Keyword: int, require 4 bytes of memory space and range from -231 to 231 - 1.

Character. Storing characters.
Keyword: char, require 1 byte of memory space and range from -27 to 27 - 1 or 0 to 28.

Boolean. Storing boolean or logical values.
Keyword: bool, is either true or false, range from 0 to 1 accordingly. Set to 0 by default.

Floating point. Storing single-precision floating-point values or decimal values.
Keyword: float, require 4 bytes of memory space.

Double floating point. Storing double-precision floating-point values or decimal values.
Keyword: double, require 8 bytes of memory space.

Void. Represents a valueless entity.
Keyword: void, used for those function which does not return a value.

Wide character. Is also a character data but has a size greater than the normal 8-bit character data type.
Keyword: wchar_t, generally 2 or 4 bytes long.
